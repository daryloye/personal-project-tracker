# AI Chat Session Logs

Place exported AI chat logs for this project in this folder.

Suggested naming format:

```text
YYYY-MM-DD-tool-topic.md
```

Example:

```text
2026-07-06-codex-project-planning.md
```

Each log should include:

- Date and tool used.
- Goal of the session.
- Important prompts.
- Useful AI outputs.
- Human review notes.
- Final decision or change made.
